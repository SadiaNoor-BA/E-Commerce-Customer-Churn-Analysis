# E-Commerce-Customer-Churn-Analysis
Project Overview
This project analyzes a dataset of 5,630 e-commerce customers to identify key drivers of churn and provide actionable retention strategies.

Key Insights
Overall Churn Rate: 16.84%.

Primary Churn Driver: Customers who raised a Complain showed a significantly higher churn risk.

Category Risk: The Mobile Phone and Mobile categories experience the highest churn rates (~27%).

Logistics Impact: Analysis suggests a correlation between Warehouse-to-Home distance and customer attrition.

Technical Skills Applied
Data Cleaning: Handled missing values for Tenure and Warehouse distance in Power Query.

DAX Measures: Created complex measures for Churn Rate %, Churned Customers, and Total Customers.

Data Visualization: Built an interactive dashboard featuring Scatter Plots, Bar Charts, and a High-Priority Retention List.
